# models 目录说明

职责：存放模型/权重及相关推理资源。

- `flux/`：图像/视频生成相关资源（具体内容视项目配置而定）。
- `gpt-sovits/`：语音合成/转换相关资源。

使用约定：
- 模型加载/调用逻辑应集中在封装层（如 `core/api/llm.py` 或专门的 service），避免在 UI/节点中直接操作文件或权重。
- 引入新模型资源时，请在本 README 说明用途、依赖与获取方式，并更新 `docs/standards/engineering-standards.md` 中的模型部分（如需）。
- 大文件/私有权重不应直接提交仓库，优先使用可配置的下载/挂载方式。
